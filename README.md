# iNC-Mill – Technical Support / 技术支持

---

## About iNC-Mill / 关于 iNC-Mill

**iNC-Mill** is a professional NC/G-code editor and toolpath viewer for CNC milling.  
It focuses on local file editing, syntax-aware viewing, and toolpath inspection.

**iNC-Mill** 是一款面向数控铣削的专业 **NC/G-code 本地编辑与刀路查看工具**，支持：
- 在本地打开和编辑 NC/G-code 文件  
- 高亮显示与缩进优化，便于阅读与修改  
- 刀路预览与基本轨迹检查

## Frequently Asked Questions / 常见问题

### 1. How do I import NC/G-code files?  
**EN:** In the app, open the *Files* tab and choose **Import File**, then select your `.nc`, `.tap`, `.gcode`, or `.txt` file or so on.  
**CN：** 在 App 中进入 **「文件」** 标签，选择 **「导入文件」**，然后从系统文件中选择 `.nc`、`.tap`、`.gcode` 或 `.txt` 等文件。

---

### 2. Which NC/G-code formats are supported?  
**EN:** iNC-Mill currently focuses on common milling NC formats (such as FANUC-style G-code). Support for more controllers (e.g. Siemens, Heidenhain) will be added in future updates.  
**CN：** 目前 iNC-Mill 主要支持常见的铣削 G-code（如 FANUC 风格）。后续会逐步增加对西门子、海德汉等更多控制系统方言的支持。

---

### 3. Subscription and Free Trial / 订阅与试用

**EN：**  
iNC-Mill uses a subscription model. New users can start with a free trial period.  
After the trial ends, continued use requires an active subscription (monthly or yearly).  
You can manage or cancel your subscription in **Settings → Apple ID → Subscriptions** on your device.

**CN：**  
iNC-Mill 采用订阅制。新用户可以先开启免费试用，  
试用期结束后，如需继续使用需保持有效订阅（按月或按年）。  
你可以在设备的 **「设置 → Apple ID → 订阅」** 中管理或取消订阅。
